## Instalação

### Frontend
```Bash
$ cd frontend
$ npm i
```

### Backend

```Bash
$ cd backend
$ npm i
```

## Execução

### Frontend
```Bash
$ npm run dev
```

### Backend
```Bash
$ npm run start:dev
```
